# Virtual X-Rays of a Simplified Leg Phantom

This project simulates X-ray projection images of a simplified leg phantom
(two concentric cylinders: bone inside soft tissue), provides a GUI to explore
acquisition parameters, and computes image-quality metrics (MSE, SSIM, Gradient
Error, Gradient Entropy).
